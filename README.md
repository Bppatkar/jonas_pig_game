A simple, interactive two-player dice game built with JavaScript, HTML, and CSS. The objective of the game is to be the first player to reach 100 points. Players take turns rolling a dice and can choose to either "Hold" their score or risk losing their turn if they roll a `1`.

## Features
- Two-player gameplay.
- Roll dice to accumulate points.
- "Hold" functionality to save points and switch turns.
- Winner determination at 100 points.
- Reset functionality to start a new game.

## How to Play
1. **Roll the Dice**: Click the "Roll Dice" button to roll a dice.  
   - If the dice shows a number other than `1`, it adds to the current score.
   - If the dice shows `1`, the turn ends, and the current score resets to `0`.

2. **Hold Your Score**: Click the "Hold" button to add the current score to your total score and switch turns.

3. **Winning the Game**: The first player to reach **100 points** wins the game.

4. **Reset**: Click the "New Game" button to reset the game.

## Project Structure
- **HTML**: Structure of the game interface.
- **CSS**: Styling for the layout and animations.
- **JavaScript**: Logic for the game mechanics and interactivity.
